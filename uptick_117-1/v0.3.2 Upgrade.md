# Uptick – v0.3.2 Upgrade Announcement


Dear Uptick Validators,

We are excited to announce that the Uptick will be upgraded to v0.3.2：
- Target Block Height: 17305380
- Estimated Time: May 14, 2026, at 12:00:00 UTC
- Commit Hash: 0d2c73b410ff0745a133569bc9fa6778969f7564


## Changed

- ethermint：EIP-7702 core access.
- Add EIP-7702 SetCodeTx positive/negative test coverage.
- Enable Shanghai/Dencun/Prague in v032 and adapt Uptick to go-ethereum v1.16.


##  For Validators Running Cosmovisor

Please refer to our [Cosmovisor](https://upticknft.gitbook.io/uptick-network-documentation/guides/quickstart/cosmovisor) Setup Guide.

Please Note: Cosmovisor will perform a full backup unless you set the environment variable: UNSAFE_SKIP_BACKUP=true

```bash
# make a dir if you've not
mkdir -p $DAEMON_HOME/cosmovisor/upgrades/v0.3.2/bin

#Download the source code and build it, go 1.23.5+ is required
git clone https://github.com/UptickNetwork/uptick.git
cd uptick
git checkout v0.3.2
make install

#Copy binary files
cp $GOBIN/uptickd $DAEMON_HOME/cosmovisor/upgrades/v0.3.2/bin/

# View version should be v0.3.2
$DAEMON_HOME/cosmovisor/upgrades/v0.3.2/bin/uptickd version --long
#commit: bde718b6b1def40da9bd79cf9872b8f1ea0b2e7d
#cosmos_sdk_version: v0.50.14
#go: go version go1.23.5 linux/amd64
#name: uptick
#server_name: uptickd
#version: v0.3.2

```
