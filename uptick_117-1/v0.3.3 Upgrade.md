# Uptick – v0.3.3 Upgrade Announcement


Dear Uptick Validators,

We are excited to announce that the Uptick Mainnet will be upgraded to v0.3.3：
- Target Block Height: 18147945
- Estimated Time: July 6th, 2026, at 10:00:00 UTC
- Commit Hash: 5a730ecb3724858b9010bc51b91e7dd5630539a5


## Changed

- v0.3.3 is a patch version centered around IBC refund security. It fixes critical issues in the IBC refund process for ERC20 token transfers and NFT cross-chain (EvmIBC).


##  For Validators Running Cosmovisor

Please refer to our [Cosmovisor](https://upticknft.gitbook.io/uptick-network-documentation/guides/quickstart/cosmovisor) Setup Guide.

Please Note: Cosmovisor will perform a full backup unless you set the environment variable: UNSAFE_SKIP_BACKUP=true

```bash
# make a dir if you've not
mkdir -p $DAEMON_HOME/cosmovisor/upgrades/v0.3.3/bin

#Download the source code and build it, go 1.23.5+ is required
git clone https://github.com/UptickNetwork/uptick.git
cd uptick
git checkout v0.3.3
make install

#Copy binary files
cp $GOBIN/uptickd $DAEMON_HOME/cosmovisor/upgrades/v0.3.3/bin/

# View version should be v0.3.3
$DAEMON_HOME/cosmovisor/upgrades/v0.3.3/bin/uptickd version --long
#commit: 5a730ecb3724858b9010bc51b91e7dd5630539a5
#cosmos_sdk_version: v0.50.14
#go: go version go1.23.5 linux/amd64
#name: uptick
#server_name: uptickd
#version: v0.3.3

```
